TROUBLE FINDING YOUR APPLICATION?

Your application may have expired. If the terms and conditions of a Licence Application are not accepted within 30 days, the application will automatically expire and you will have to start a new Licence Application.
For security purposes, you need to be logged in as the same type of user that the Licence Application made on behalf of, i.e. if the application was made on behalf of a company you need to be logged in to SFOS as a Company (or Partner) user in order to retrieve it. Of, if the application was made for a sole applicant, then you need to be logged in as an Individual (or Partner) user to be able to retrieve it.
You are currently logged in as an Individual user.
If the ARN you entered is for a Company application, please select Logout from the Main Menu and then re-login with the company Username and Password. You may then retrieve the Company Application using the Application Reference Number (ARN) and PIN sent to the Company email address you specified in your Licensing Information.
You are currently logged in as a COMPANY user.
If the ARN you entered is for an Individual (Sole Applicant) application, please select Logout from the Main Menu and then re-login with an Individual Username and Password. You may then retrieve the Individual's Application using the Application Reference Number (ARN) and PIN sent to the email address you specified in your Licensing Information.
